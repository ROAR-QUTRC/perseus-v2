## Fast-LIO

This package has git submodules, which can't be retrieved by fetchFromGitHub
automatically. This has to be overridden from the generated package. Setting
fetchSubmodules to true just uses fetchGit instead, so I've used fetchGit to be
cleaner. Fast-LIO also compiles specifically for c++14, which conflicts with the
rclcpp library, which is written for c++17.

## Lidarslam-msgs

This package has ament-cmake-auto missing from its package.xml, so the
autogenerated nix package also doesn't contain the dependency.

## Ndt-omp-ros2

This package includes pcl/filters/boost.h, which has been removed as it was just
including header files from the Boost package. It also has an older cpp
standard, which is unsupported by pcl. These have been changed to reference all
of the boost headers and cpp version 17 instead.

## Graph based slam

This package has nav_msgs and std_msgs missing from its package.xml, so the
autogenerated nix package also doesn't contain the dependencies.
